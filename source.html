Source Code

To get the visualization running locally on your computer,
the first thing you need to do is follow this link.
This should take you to the github repository that houses all of our source code.
To obtain all of it at once, simply hit "Download Zip" on the right side of the page.
Once this is done, you can extract all of the files onto your computer. From here,
opening the "simpson.html" file onto the browser of your choice should run the visualization.
And there you have it! The visualization is now running locally.
